# Lab4Tech LLM RAG Projects

This repository contains the HireReadyAI project and its early raw prototype.

## Main Project

For the full project details, architecture, setup instructions, features, motivation, and  summary, open:

```text
HireReadyAI/README.md
```

`HireReadyAI/` is the current full-stack application.

It includes:

- CV parsing and skill extraction
- job description extraction using Mistral
- weighted CV-job matching
- personalized learning path generation
- company-context-aware interview generation
- interview preparation mode
- adaptive interview mode
- LLM-based scoring
- Python coding question execution
- RAG-grounded question generation and evaluation
- FAISS vector index lifecycle: use existing, update, recreate
- React frontend and FastAPI backend

HireReadyAI-raw-prototype/README.md

`HireReadyAI-raw-prototype/` contains the early Streamlit/Gemini prototype used to explore adaptive interview and learning-assistant ideas before building the full application.

## Project Motivation

HireReadyAI was built to help candidates understand how well their CV fits a target role, identify missing skills, practise realistic interview questions, receive scored feedback, and follow a personalized learning path.

The project explores how LLMs, RAG, vector databases, skill matching, scoring rubrics, and adaptive learning can work together in a practical AI career-preparation product.
